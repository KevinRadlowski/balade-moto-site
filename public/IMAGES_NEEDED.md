# Images nécessaires pour le site

Pour que le site fonctionne correctement, veuillez ajouter les images suivantes dans le dossier `public/` :

## Images requises

### 1. Logo principal
- **Fichier** : `logo.png`
- **Taille recommandée** : 512x512px (carré)
- **Format** : PNG avec transparence
- **Usage** : Header, Footer, Hero Section

### 2. Image de fond Hero Section
- **Fichier** : `bg-home.png`
- **Taille recommandée** : 1920x1080px minimum
- **Format** : JPG
- **Usage** : Arrière-plan de la section hero (page d'accueil)
- **Suggestion** : Photo de moto et/ou voiture en action, paysage de route

## Images optionnelles (pour la section Screenshots)

### 3. Screenshot 1 de l'application
- **Fichier** : `mockups/screen1.png`
- **Taille recommandée** : 1080x1920px (format mobile portrait)
- **Format** : PNG
- **Usage** : Section "Découvrez l'application" - Interface principale

### 4. Screenshot 2 de l'application
- **Fichier** : `mockups/screen2.png`
- **Taille recommandée** : 1080x1920px (format mobile portrait)
- **Format** : PNG
- **Usage** : Section "Découvrez l'application" - Organisation de balades

## Notes

- Si une image n'est pas disponible, le site affichera un placeholder avec un message
- Les images seront automatiquement optimisées par Next.js
- Utilisez des images de bonne qualité pour un rendu optimal
- Compressez les images avant de les ajouter pour améliorer les performances
